# Data

Auto-generated JSON files extracted from `wube/factorio-data`.

Populated by the build pipeline (not yet implemented). Expected files:

- `recipes.json` — all recipes with ingredients, outputs, crafting times
- `entities.json` — all entity stats (HP, speed, power, pollution)
- `technologies.json` — full technology tree with prerequisites and costs
- `modules.json` — module effects and beacon distribution
- `surfaces.json` — planet properties (Space Age)

Tagged with Factorio version number for traceability.
