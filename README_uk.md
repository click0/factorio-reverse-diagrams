# Factorio Reverse Diagrams

> Відображення та переклад правил симуляції Factorio у вигляді інтерактивних дослідницьких пояснень.

Натхненно [зворотними діаграмами SimCity Хаїма Гінгольда (2016)](assets/reference/SimCity_Reverse_Diagrams_Gingold_2016.pdf) та традицією дослідницьких пояснень [Брета Віктора](https://worrydream.com/), [Бартоша Цєхановського](https://ciechanow.ski/), [Red Blob Games](https://www.redblobgames.com/) та [Нікі Кейса](https://ncase.me/).

---

## Що це?

Комплексна документація зворотного інжинірингу **симуляції Factorio** (база 2.0 + розширення Space Age), представлена як:

- **Інтерактивні веб-розділи** — текстові пояснення з вбудованими живими симуляціями
- **Вбудовувані віджети** — окремі інтерактивні діаграми для форумів, Reddit, вікі
- **PDF-постери** — друковані великоформатні архівні плакати
- **Відкриті дані** — JSON-файли, витягнуті з прототипів гри

Доступно **англійською** та **українською**.

## Статус: 44 інтерактивні діаграми (v0.50)

Усі заплановані інтерактивні діаграми реалізовані в 11 тематичних секціях — від ядра симуляції та механіки конвеєрів до якості Space Age та калькуляторів у стилі cheat sheet.

## Вбудовування на ваш сайт

Будь-яку діаграму можна вбудувати на зовнішній сайт через iframe. Сторінки вбудовування відображають **лише віджет** — без бічної панелі, заголовка чи навігації.

### Формат URL

```
https://click0.github.io/factorio-reverse-diagrams/#/embed/{widget-id}
```

### Приклад: вбудувати механіку конвеєрів

```html
<iframe
  src="https://click0.github.io/factorio-reverse-diagrams/#/embed/belt-simulator"
  width="800"
  height="500"
  frameborder="0"
  style="border: 1px solid #2a2a4e; border-radius: 8px;"
></iframe>
```

### Примітки

- Віджети автоматично успадковують темну тему Factorio
- Мову можна встановити через URL: додайте `?lng=uk` для української
- Усі інтерактивні елементи (відтворення/пауза, слайдери) працюють всередині iframe
- Немає X-Frame-Options чи CSP заголовків — вбудовування необмежене
- Кожен віджет завантажується лише при потребі (2–10 КБ gzip на віджет)
